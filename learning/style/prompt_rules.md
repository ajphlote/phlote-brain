# Prompt Rules

## Always
- Start faster — first line earns attention or loses it
- Strong single opening statement
- Short thought blocks with breathing room between them
- Make complex ideas feel inevitable
- Write like a realization, not a pitch

## Never
- Sound preachy or lecture-y
- Sound like marketing copy
- Use buzzwords when a simpler phrase exists
- Explain the same point twice
- Open with "In a world where..." or "The future of..."

## Formatting
- Single line breaks between short thoughts
- Double line break between sections
- No bullet points in Twitter or Instagram
- No em dashes in casual posts

## Voice Check
Ask: does this sound like a trusted friend explaining something?
If it sounds like a press release — rewrite it.
If it sounds like a prediction — make it an observation instead.
If it's longer than it needs to be — cut it.
